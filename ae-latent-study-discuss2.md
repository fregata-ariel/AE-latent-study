# AE Latent Study Discuss 2

## Lattice Step 2 と Discuss 1 の比較

`ae-latent-study-discuss1.md` の中心的な整理は、次の 2 点だった。

- 潜在空間そのものは保型形式ではなく、まずは群作用で割った商空間の座標として捉えるべきである
- 保型形式は、その潜在空間上の自然な関数系や基底として現れうる

今回の lattice 実験は、この見取り図にかなり素直に沿っている。
入力データは格子テータ関数であり、背後には `SL₂(Z)` による同値関係と楕円曲線のモジュライ的構造がある。
そのため、この研究は「一般データへの比喩」ではなく、最初から保型的・モジュライ的な文脈に入っている。

## Step 1 / Step 2 が示したこと

Step 1 の結果は、再構成だけでは商空間構造は自動的には現れないことを示した。

- 再構成誤差はほぼ 0 だった
- しかし latent はほぼ 1 次元曲線に退化した
- `SL₂(Z)` 同値点の latent 距離も大きかった

これは `ae-latent-study-discuss1.md` の「標準 AE/VAE は群や商空間を明示的に組み込んでいない」という注意と一致している。

Step 2 では状況が大きく改善した。

- max 正規化で振幅スケールの支配を弱めた
- `log10|j|` を使う評価に切り替えた
- modular invariant loss を入れて `E(gx) ≈ E(x)` を明示的に促した

その結果、採用 run `lattice_standard_norm_inv` では

- reconstruction MSE は十分小さいまま
- `max Spearman(log10|j|) = 0.9825`
- `mean modular distance = 0.000963`

まで改善した。

これは、「AE を `X -> X/G -> X` の近似として見る」という見方に対する、かなり強い実証になっている。

## 現在の位置づけ

ただし、現在地は慎重に言い表した方がよい。

今やれているのは主に次のことだ。

- 同値類に属する入力を近い latent に送る
- latent 上で `j` のような観測量を安定に読む
- 商空間らしい表現を、学習によってある程度作る

まだやれていないのは次のことだ。

- latent 自体を 2 次元商空間の忠実な座標として再現すること
- `E(gx) = ρ(g)E(x)` 型の共変表現を持つこと
- `D(ρ(g)z) = g·D(z)` のような decoder 側の整合性を持つこと
- latent 上の自然基底や固有関数を調べ、保型形式との対応を論じること

つまり今の研究は、

- 「保型形式の比喩」よりは明らかに前に進んでいる
- しかし「保型形式を latent 上の自然基底として扱う段階」にはまだ入っていない

という位置にある。

より短く言えば、今は

**群作用付き商空間学習の第一段階に成功した**

のであって、

**保型形式そのものを学習したわけではない**

ということになる。

## ここまでで得た重要な教訓

### 1. 不変性は自然には出てこない

標準 AE をそのまま回しても、再構成は成功するが商空間構造は崩れる。
したがって、対称性は loss や architecture に明示的に入れる必要がある。

### 2. 今の成功は「商空間の潰し込み」に強い

`lattice_standard_norm_inv` は modular distance をほぼ 0 まで下げた。
これは orbit collapse としては非常に良い。

### 3. しかし「2D の座標化」はまだ弱い

最良 run の latent scatter も、まだかなり細いリボン状である。
これは、同値類の貼り合わせには成功していても、商空間全体の幾何を豊かに展開できていないことを示唆する。

### 4. VAE はヒントをくれるが答えではない

VAE はより 2D 的な広がりを示したが、再構成と `log10|j|` 相関を落とした。
これは、単純なユークリッド Gaussian prior がこの幾何に必ずしも合っていない可能性を示すが、同時に「幾何を変えるだけでは足りない」ことも示している。

## 今後の方針

今後は、次の順で進めるのがよい。

### Phase A. 商空間学習としての精密化

まずは `lattice_standard_norm_inv` を基準系に固定し、
「modular distance が小さい」だけではなく
「商空間の 2D 座標としてどれだけ良いか」を測る指標を導入する。

候補:

- latent の局所有効次元
- 近傍保存性
- 基本領域上の点が latent でどれだけ 2D に展開されるか
- `j` の 2 実次元情報をどう保持しているか

今の評価は `log10|j|` にかなり依存しているので、1 次元的に良い潜在表現でも高得点を取りうる。
したがって、次の課題は「不変性の達成」から「商空間座標の品質評価」へ軸を移すことになる。

### Phase B. Invariant AE から Equivariant AE へ

次に、`E(gx) ≈ E(x)` だけでなく
`E(gx) = ρ(g)E(x)` 型の表現を試す。

方向性としては 2 つある。

- invariant な成分と gauge 的な成分を分ける
- `S`, `T` に対応する明示的な latent action `ρ(S), ρ(T)` を学習する

これができると、decoder 側にも
`D(ρ(g)z) ≈ g·D(z)`
の整合性を課せるようになる。

ここで初めて、Discuss 1 で整理した「E/D は群作用と整合する写像」という図式に真正面から入れる。

### Phase C. Latent 上の関数空間へ進む

その次に初めて、

- latent 上の観測量
- latent graph 上のラプラシアン
- 固有関数や自然基底

を考える段階に入る。

ここでの問いは、

- `j`
- theta 系の特徴量
- 生成密度

のような関数が、latent 上でどのような基底展開を持つかである。

この段階になって初めて、

**保型形式や Maaß form を「潜在空間上の自然基底」として比較する**

という本来の動機づけが、研究として具体的になる。

### Phase D. 非ユークリッド latent はその後

双曲空間や商空間 latent は魅力的だが、今の結果を見る限り、

- geometry だけ変えても不十分
- group action を入れないと本筋に乗らない

ので、優先順位としては invariant/equivariant の設計の後に置くのがよい。

## 当面の研究上の立場

この研究の現時点での最も安全で強い主張は、次のようになる。

> 格子テータ関数のようなモジュライ的データに対しては、
> autoencoder は適切な正規化と群作用ベースの損失を入れることで、
> `SL₂(Z)` の同値類を尊重した潜在表現を学習できる。

一方で、次の主張はまだ早い。

> 潜在空間そのものが保型形式である
> あるいは、保型形式の理論がそのまま latent の理論として完成している

したがって、今後の自然な進め方は

1. 商空間学習としての品質を高める
2. 共変表現を導入する
3. その上で latent 上の関数空間を調べる

という順序になる。

## 一言で言えば

この研究は今、

**「保型形式の比喩」から「商空間学習の実証」へ進んだ段階**

にある。

次の目標は、

**「商空間学習の実証」から「latent 上の自然基底の解析」へ進むこと**

である。
